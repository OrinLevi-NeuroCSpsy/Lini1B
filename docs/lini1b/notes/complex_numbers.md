# מספרים מרוכבים

!!! info "מה נלמד בפרק זה?"
    המספרים המרוכבים מרחיבים את הממשיים ומאפשרים לפתור משוואות כמו $x^2 + 1 = 0$. נלמד על פעולות חשבון, הצגות שונות ושורשים מרוכבים.

---

## הגדרת המספרים המרוכבים

!!! abstract "הגדרה"
    **המספרים המרוכבים** הם הקבוצה:
    $$\mathbb{C} = \{a + bi \mid a, b \in \mathbb{R}\}$$

    כאשר $i$ מקיים $i^2 = -1$.

    - $a = \text{Re}(z)$ נקרא **החלק הממשי**
    - $b = \text{Im}(z)$ נקרא **החלק המדומה**

---

## פעולות חשבון

### חיבור וחיסור

$$(a + bi) + (c + di) = (a + c) + (b + d)i$$
$$(a + bi) - (c + di) = (a - c) + (b - d)i$$

### כפל

$$(a + bi)(c + di) = ac + adi + bci + bdi^2 = (ac - bd) + (ad + bc)i$$

### חילוק

$$\frac{a + bi}{c + di} = \frac{(a + bi)(c - di)}{(c + di)(c - di)} = \frac{(ac + bd) + (bc - ad)i}{c^2 + d^2}$$

!!! example "דוגמה"
    חשבו $\displaystyle\frac{3 + 2i}{1 - i}$:

    $$\frac{3 + 2i}{1 - i} = \frac{(3 + 2i)(1 + i)}{(1 - i)(1 + i)} = \frac{3 + 3i + 2i + 2i^2}{1 - i^2} = \frac{3 + 5i - 2}{1 + 1} = \frac{1 + 5i}{2}$$

---

## הצמוד והערך המוחלט

### הצמוד

!!! abstract "הגדרה"
    **הצמוד** של $z = a + bi$ הוא $\bar{z} = a - bi$

**תכונות הצמוד:**

| תכונה | נוסחה |
|-------|-------|
| צמוד של סכום | $\overline{z + w} = \bar{z} + \bar{w}$ |
| צמוד של מכפלה | $\overline{z \cdot w} = \bar{z} \cdot \bar{w}$ |
| צמוד של מנה | $\overline{z / w} = \bar{z} / \bar{w}$ |
| צמוד של צמוד | $\overline{\bar{z}} = z$ |
| מספר ממשי | $z \in \mathbb{R} \Leftrightarrow z = \bar{z}$ |
| מספר מדומה טהור | $z$ מדומה טהור $\Leftrightarrow z = -\bar{z}$ |

**נוסחאות שימושיות:**

$$\text{Re}(z) = \frac{z + \bar{z}}{2}, \quad \text{Im}(z) = \frac{z - \bar{z}}{2i}$$

### הערך המוחלט (מודולוס)

!!! abstract "הגדרה"
    **הערך המוחלט** של $z = a + bi$ הוא:
    $$|z| = \sqrt{a^2 + b^2} = \sqrt{z \cdot \bar{z}}$$

**תכונות הערך המוחלט:**

| תכונה | נוסחה |
|-------|-------|
| אי-שליליות | $|z| \geq 0$, ו-$|z| = 0 \Leftrightarrow z = 0$ |
| מכפלה | $|z \cdot w| = |z| \cdot |w|$ |
| מנה | $|z / w| = |z| / |w|$ |
| אי-שוויון המשולש | $|z + w| \leq |z| + |w|$ |
| קשר לצמוד | $z \cdot \bar{z} = |z|^2$ |

## הצגה במישור גאוס

<div class="math-box" markdown>

- **ציר $x$** - הציר הממשי
- **ציר $y$** - הציר המדומה

## הצגה פולרית (טריגונומטרית)

!!! abstract "הגדרה"
    כל מספר מרוכב $z \neq 0$ ניתן לכתוב בצורה:
    $$z = r(\cos\theta + i\sin\theta) = r \cdot \text{cis}(\theta)$$

    כאשר:

    - $r = |z|$ הוא **המודולוס**
    - $\theta = \arg(z)$ הוא **הארגומנט** (הזווית עם ציר $x$ החיובי)

### המרה בין הצגות

**מקרטזית לפולרית:**

$$r = \sqrt{a^2 + b^2}, \quad \theta = \arctan\left(\frac{b}{a}\right)$$

!!! warning "שימו לב לרביע!"
    יש להתאים את $\theta$ לרביע שבו נמצא $z$

**מפולרית לקרטזית:**

$$a = r\cos\theta, \quad b = r\sin\theta$$

---

## כפל וחילוק בצורה פולרית

!!! note "משפט: כפל וחילוק"
    אם $z_1 = r_1 \text{cis}(\theta_1)$ ו-$z_2 = r_2 \text{cis}(\theta_2)$:

    **כפל:**
    $$z_1 \cdot z_2 = r_1 r_2 \cdot \text{cis}(\theta_1 + \theta_2)$$

    **חילוק:**
    $$\frac{z_1}{z_2} = \frac{r_1}{r_2} \cdot \text{cis}(\theta_1 - \theta_2)$$

---

## נוסחת דה-מואבר

!!! note "משפט: דה-מואבר"
    לכל $n \in \mathbb{Z}$:
    $$(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$$

    או בקיצור:
    $$[\text{cis}(\theta)]^n = \text{cis}(n\theta)$$

!!! example "דוגמה"
    חשבו $(1 + i)^{10}$:

    1. נמיר לצורה פולרית: $1 + i = \sqrt{2} \cdot \text{cis}(45°)$
    2. נשתמש בדה-מואבר:
    $$(1 + i)^{10} = (\sqrt{2})^{10} \cdot \text{cis}(450°) = 32 \cdot \text{cis}(90°) = 32i$$

---

## שורשים מרוכבים

!!! note "משפט: שורשים מסדר $n$"
    למספר מרוכב $z = r \cdot \text{cis}(\theta)$ יש בדיוק $n$ שורשים מסדר $n$:

    $$w_k = \sqrt[n]{r} \cdot \text{cis}\left(\frac{\theta + 2\pi k}{n}\right), \quad k = 0, 1, \ldots, n-1$$

**תכונות השורשים:**

- $n$ השורשים מפוזרים באופן שווה על מעגל ברדיוס $\sqrt[n]{r}$
- הזווית בין שורשים סמוכים היא $\displaystyle\frac{2\pi}{n}$

!!! example "דוגמה: שורשים של $1$"
    **שורשים שלישיים של 1** ($z^3 = 1$):

    $$w_0 = \text{cis}(0°) = 1$$
    $$w_1 = \text{cis}(120°) = -\frac{1}{2} + \frac{\sqrt{3}}{2}i$$
    $$w_2 = \text{cis}(240°) = -\frac{1}{2} - \frac{\sqrt{3}}{2}i$$

---

## שורשים של מספרים שליליים

!!! example "דוגמה"
    מצאו את $\sqrt{-4}$:

    $$\sqrt{-4} = \sqrt{4 \cdot (-1)} = \sqrt{4} \cdot \sqrt{-1} = 2i$$

    **שורשים ריבועיים של $-4$:**

    $-4 = 4 \cdot \text{cis}(180°)$, לכן:

    $$w_0 = 2 \cdot \text{cis}(90°) = 2i$$
    $$w_1 = 2 \cdot \text{cis}(270°) = -2i$$

---

## טיפים לפתרון

<div class="tip-box" markdown>

1. **בחילוק** - תמיד הכפילו במצומד של המכנה

2. **בחזקות גבוהות** - עברו להצגה פולרית והשתמשו בדה-מואבר

3. **בשורשים** - זכרו שיש $n$ שורשים מסדר $n$, והם מפוזרים באופן שווה על מעגל

4. **בזיהוי ארגומנט** - שימו לב לרביע! $\arctan$ נותן תשובה בין $-90°$ ל-$90°$ בלבד

5. **זהות שימושית:** $e^{i\theta} = \cos\theta + i\sin\theta$ (נוסחת אוילר)

</div>
